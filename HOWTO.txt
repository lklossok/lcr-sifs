### How to use:


## Build Dockerfile
    # 1. Open Docker Desktop
    # 2. Build image from Dockerfile lcr-sifs/dockerfiles/dlbclone/Dockerfile
        {bash}
        cd lcr-sifs/dockerfiles/dlbclone
        docker build -t lklossok/dlbclone:latest .
    # 3. Publish to Docker Hub
        {bash}
        docker push lklossok/dlbclone:latest


## Locally build the Singularity image on thanos OR gphost:
    # 1. conda activate base
    # 2. Build Singularity image from Docker Hub
        {bash}
        singularity build lcr-sifs/sifs/dlbclone/dlbclone.sif docker://lklossok/dlbclone:latest
    # 3. Test Singularity image R terminal
        {bash}
        singularity exec dlbclone.sif R
        installed.packages()[,c('Package','LibPath')] # check installed R packages and their library paths
